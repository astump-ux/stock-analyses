Geschäftsmodell
Quellen ≤ 6 Monate

Drei Segmente: Productivity, Intelligent Cloud, More Personal Computing

Microsoft erwirtschaftete im Q3 FY2026 Umsätze von $82.9 Mrd. (+18% YoY). Die drei berichteten Segmente: Productivity & Business Processes ($35.0 Mrd., +13%) – inkl. M365 Commercial + Consumer, Dynamics 365, LinkedIn; Intelligent Cloud ($34.7 Mrd., +30%) – inkl. Azure (40% YoY), Server Products, GitHub, Nuance; More Personal Computing ($13.2 Mrd., +1%) – inkl. Windows, Surface, Xbox, Search/News-Advertising. Die Microsoft Cloud-Metrik (segmentübergreifend) erreichte $54.5 Mrd. (+29%), RPO $627 Mrd. (+99% YoY).

Revenue-Streams & Unit Economics

Microsoft ist die profitabelste Big-Tech-Firma: Konzern-Op-Marge Q3 FY26 erreichte 46.3% ($38.4 Mrd. Op-Income). Productivity-Segment-Op-Marge: 47%, Intelligent Cloud: 44%, MPC: ~30%. M365 Copilot (Add-On) hat eine geschätzte 85% Bruttomarge (vs. 75% für klassisches M365). Azure ist Hochmargen-Cloud (~45% Op-Marge geschätzt), Office/M365 ist Recurring-Revenue mit hoher Renewal-Rate (>95%). Die AI-Business läuft mit $37 Mrd. ARR (+123% YoY), getrieben durch Azure OpenAI Service, Copilot, und Foundry.

Strategischer Pivot: OpenAI-Restructuring + Maia-Silicon

Am 27.04.2026 wurde das OpenAI-Agreement fundamental restrukturiert: (1) Microsoft-IP-Lizenzierung von OpenAI-Modellen wird non-exclusive (vorher exklusiv) bis 2032 verlängert; (2) Microsoft zahlt KEINE Revenue-Share mehr an OpenAI; (3) OpenAI-Revenue-Share an Microsoft (~20%) läuft bis 2030 mit $38 Mrd. Cap; (4) Microsoft hält 26.79% OpenAI-Stake (Wert: ~$228 Mrd.). Parallel: Maia 200 AI-Accelerator live in Iowa/Arizona, +30% Tokens/Dollar vs. NVIDIA. Microsoft positioniert sich auf Multi-Model-Strategie (OpenAI + Anthropic Claude + Mistral + xAI Grok + Meta Llama + DeepSeek auf Azure).

Quellen: Microsoft Q3 FY26 8-K (29.04.2026); Q3 FY26 Earnings Call Transcript; SEC FY26 Q3 Filing; TradingKey OpenAI-Restructuring-Analyse (04/2026); Investing.com (12.05.2026).

Burggraben & Wettbewerbsvorteile
Score: 9/10

Enterprise-Distribution-Moat: M365 + Azure + Active Directory

Microsofts dominante Position kommt aus tiefer Enterprise-Integration: 80% der Fortune 500 nutzen M365 Copilot, ~95% der Fortune 500 nutzen Azure für mindestens einen Workload. Active Directory / Entra ID ist die Standard-Identity-Plattform der Enterprise-Welt. Diese Switching-Costs sind extrem hoch – ein Wechsel von M365 zu Google Workspace oder von Azure AD zu Okta dauert für eine 10.000-User-Org Jahre.

Multi-Model-AI-Hub: Azure als Modell-Agnostisches Layer

Q3 FY26 hat MSFT seinen AI-Moat strukturell verstärkt: Azure hostet inzwischen OpenAI + Anthropic Claude + xAI Grok + Meta Llama + Mistral + DeepSeek – das macht Azure zur universellen AI-Modell-Plattform. Copilot-Seats sprangen von 15M auf 20M (+33% QoQ), was die Adoption auf Enterprise-Scale validiert. RPO $627 Mrd. (+99% YoY) sind beispiellos – mehr als doppelt so viel wie vor einem Jahr.

Schwächen & Verwundbarkeiten

Drei Burggraben-Risse: (1) OpenAI-Abhängigkeit war strukturelles Risiko – das Restructuring hat das reduziert, aber Microsoft musste $38B-Cap akzeptieren als Preis dafür; (2) Azure-Wachstum lagged GCP – Azure +40% (Q3 FY26) ist beeindruckend, aber Google Cloud +63% wächst schneller, MSFT verliert relativ Marktanteile; (3) Gaming-Schwäche – Xbox-Hardware -30% YoY in 2 aufeinanderfolgenden Quartalen, Activision-Integration nicht überzeugend monetarisiert.

Quellen: MSFT Q3 FY26 Earnings Call (29.04.2026); TechCrunch OpenAI-Restructuring (27.04.2026); Synergy Research Cloud Market Share Q1 2026; TECHi MSFT Analysis (04/2026).

🐂 Das Bullische Narrativ

Säule 1 – Azure Reacceleration ist real: Nach 2 Quartalen Deceleration (Q1: 40% → Q2: 39%) sprang Azure in Q3 FY26 zurück auf +40% YoY (39% CC). Capacity Constraints lösen sich. AI-Run-Rate $37B (+123% YoY) ist das schnellste skalierende Geschäft in Big-Tech. RPO $627 Mrd. (+99% YoY) liefert Multi-Jahres-Visibility.

Säule 2 – Copilot validiert als Enterprise-Standard: Paid-Seats von 15M → 20M (+33% QoQ), präsent bei 80% der Fortune 500. ARPU stabil trotz Volumen-Verhandlungen. Guggenheim-Schätzung: Copilot kann bis +30% M365-Revenue-Uplift bei ~100% inkrementellen Margen liefern. Das ist eine multi-Hundert-Milliarden-Opportunity bei 47% Productivity-Op-Marge.

Säule 3 – Bewertung im Ausnahme-Diskount + OpenAI-Hidden-Value: P/E 24x ist 22% unter 10J-Schnitt – seltenes Setup. Plus: $228 Mrd.
